# Troubleshooting Guide

## Login Issues

If you cannot log in, try the following steps:

1. Clear your browser cache and cookies
2. Reset your password via the "Forgot Password" link
3. Ensure your account has not been deactivated (contact support)

## Sync Problems

Documents not syncing? Check these common causes:

- Ensure you have a stable internet connection
- Check if the service status page shows any outages
- Try logging out and back in to refresh your session token

## Performance Issues

If the application is running slowly:

- Close unnecessary browser tabs
- Disable browser extensions that may interfere
- Try using a different browser (Chrome or Firefox recommended)
- Contact support if issues persist for more than 24 hours
