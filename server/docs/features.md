# Product Features

## Core Features

- Real-time collaboration: Multiple users can edit documents simultaneously
- Version history: Every change is tracked with full rollback capability
- Smart search: AI-powered search across all your documents
- Export options: PDF, Word, HTML, and Markdown export

## Premium Features

- Custom branding: Add your company logo and colors
- API access: Full REST API for programmatic access
- Priority support: 24/7 dedicated support team
- Advanced analytics: Usage statistics and engagement metrics

## Integrations

- Slack: Get notifications and share documents directly
- GitHub: Sync documentation with your repositories
- Jira: Link documents to tickets and sprints
